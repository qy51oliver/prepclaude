# PrepClaude

AI-powered interview prep for any role — not just software engineering.

**[Try it → prepclaude.vercel.app](https://prepclaude.vercel.app)**

---

Paste a job description, get 10 tailored interview questions, practice your answers, and get scored feedback from Claude. Works for any job: software, medical, finance, research, whatever.

## How it works

1. Paste any job description
2. Optionally upload your resume for personalized questions
3. Get 10 questions specific to that role and company
4. Answer each one — Claude scores you and shows what a strong answer looks like
5. See your weak areas summary when you're done

## Stack

React · TypeScript · Vite · Claude API · Vercel

## Run locally

```bash
git clone https://github.com/qy51oliver/prepclaude
cd prepclaude
npm install
npm run dev
```

Get a free API key at [console.anthropic.com](https://console.anthropic.com). Your key is never stored — it only lives in your browser session.
